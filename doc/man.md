## web api

#### 接口

##### 1. 单个服务器操作

http://localhost:5000/gs

| 参数 | 说明                        |
| ---- | --------------------------- |
| id   | 服务器id / 服务器目录名索引 |
| cmd  | 执行的**命令**              |

支持的**命令**：

| 命令      | 说明       |
| --------- | ---------- |
| create    | 创建服务器 |
| delete    | 删除服务器 |
| start     | 开启服务器 |
| hotUpdate | 热更新     |
| exit      | 关闭服务器 |

##### 2. 获取运行中服务器列表

http://localhost:5000/gs_running
