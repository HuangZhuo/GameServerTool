# GameServerTool
传奇游戏服多服运维工具

![main_interface](https://s2.loli.net/2022/09/07/pjRS73Oco6QAlfU.png)

## [文档](https://gitee.com/JoeyHw/GameServerTool/wikis/pages)

## 功能
- 批量创建服务器，支持创建脚本和配置文件自定义规则
- 批量开启，关闭，重启，更新服务器
- 支持计划任务，定时操作服务器
- 支持通过web接口操作服务器，和游戏后台管理系统配合使用

## 打包
run `build.bat`, python3 only!
