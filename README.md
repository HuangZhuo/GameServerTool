# GameServerTool
传奇游戏服多服运维工具

## 功能
- 批量创建服务器，支持创建脚本和配置文件自定义规则
- 批量开启，关闭，重启，更新服务器
- (V3.3+)支持计划任务，定时操作服务器

## 版本历史
### V3.4
提供web接口
### V3.3
服务器计划任务功能
### V3.2
~202191前线上服役版本

## 打包EXE
run `build.bat`, python3 only!
